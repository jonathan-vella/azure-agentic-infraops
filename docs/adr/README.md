## About ADRs

ADRs capture important architectural decisions made along with their context and consequences.
They provide a historical record of why certain design choices were made.

## Creating New ADRs

Use the `adr-generator` custom agent to create new ADRs:

1. Press `Ctrl+Shift+A` in VS Code
2. Select `adr-generator`
3. Describe the decision you need to document
